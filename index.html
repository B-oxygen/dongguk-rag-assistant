04. Prompt Engineering

환각(Hallucination) 방지 기법

01
모르면 솔직하게
"추측하지 말고 '알 수 없습니다'라고 답하세요."
02
근거 제시 의무화
"주장에는 반드시 배경 데이터를 함께 기술하세요."
03
사실과 추론 분리
"확인된 사실과 논리적 추론을 구분하여 서술하세요."
04
출처 명시 강제
"답변의 근거가 된 도서, 논문, 공식 문서를 제시하세요."
05
보수적 표현 사용
"불확실하면 '가능성이 있습니다'로 표현하세요."
06
범위 제한 설정
"제공된 문서 안의 정보로만 답변하세요."

08. LangChain Workflow

LangChain RAG 전체 처리 흐름

1
Document Loader
PDF, 웹페이지 등에서 원시 데이터를 추출
2
Text Splitter
문서를 검색에 적합한 크기의 청크로 분할
3
Embedding
텍스트 조각을 고차원 벡터로 변환
4
Vector Store
벡터와 메타데이터를 FAISS 등에 저장
5
Retriever
유사도가 높은 문서 조각을 검색
6
LLM 생성
문서 원문을 컨텍스트에 결합해 정답 도출

10. Frontend Development

Streamlit: 초고속 AI 웹앱 빌더

HTML/CSS 없이 Python 코드만으로 인터랙티브한 웹 앱 구현 및 배포

app.py
import streamlit as st st.title("나의 AI 비서") user_input = st.text_input( "질문을 입력하세요:" ) if st.button("답변 생성"): with st.spinner("생각 중..."): # LLM 호출 로직 answer = get_answer(user_input) st.write(answer)
